## 🌍 لماذا هذا المشروع؟

أكثر من **70 مليون شخص أصم** حول العالم يواجهون حواجز تواصل يومية. المنظمات غير الربحية التي تخدم هذه المجتمعات تحتاج إلى أدوات **مجانية، تعمل بدون إنترنت، وسهلة الاستخدام**.

**SingAI** يحل هذه المشكلة بتحويل الكلام أو النص إلى لغة إشارة مرئية عبر أفاتار متحرك — كل المعالجة تتم على الجهاز مباشرة.

## 🏗️ بنية المشروع

يعتمد المشروع على **Clean Architecture** مع فصل واضح بين الطبقات:

```
lib/
├── 📁 core/              # مكونات أساسية مشتركة
│   ├── constants/        # ثوابت التطبيق والألوان والنصوص
│   ├── errors/           # معالجة الأخطاء والاستثناءات
│   ├── network/          # خدمات الشبكة والاتصال
│   └── utils/            # أدوات مساعدة (logger, validators)
│
├── 📁 domain/            # طبقة النطاق (قواعد العمل)
│   ├── models/           # نماذج البيانات
│   ├── repositories/     # واجهات المستودعات
│   └── use_cases/        # حالات الاستخدام
│
├── 📁 application/       # طبقة التطبيق (الخدمات)
│   ├── stt/              # خدمة تحويل الكلام لنص
│   ├── normalization/    # خدمة تطبيع النص
│   ├── sign_mapping/     # خدمة تعيين الإشارات
│   ├── avatar/           # خدمة عرض الأفاتار
│   ├── voice_recording/  # خدمة تسجيل الصوت
│   ├── export/           # خدمة التصدير والمشاركة
│   ├── asset_manager/    # إدارة النماذج والأصول
│   └── cache/            # إدارة التخزين المؤقت
│
├── 📁 presentation/      # طبقة العرض (الواجهة)
│   ├── pages/            # صفحات التطبيق
│   ├── widgets/          # مكونات الواجهة
│   └── bloc/             # إدارة الحالة (Cubit/BLoC)
│
└── 📁 config/            # إعدادات وحقن التبعيات
```

### تدفق البيانات

```
🎙️ صوت / ✍️ نص
      ↓
🧠 STT (Whisper محلي)
      ↓
📝 تطبيع النص (RAG محلي)
      ↓
🤟 تعيين الإشارات
      ↓
🧑‍💻 عرض الأفاتار
      ↓
💾 تخزين مؤقت محلي
```

## 🛠️ التقنيات المستخدمة

### الإطار الأساسي
- **Flutter 3.24+** — تطبيق متعدد المنصات (Android & iOS)
- **Dart 3.4+** — لغة البرمجة

### إدارة الحالة
- **flutter_bloc** — إدارة حالة التطبيق
- **freezed** — نماذج بيانات غير قابلة للتغيير

### الذكاء الاصطناعي المحلي
- **flutter_whisper** — تحويل الكلام لنص محلياً
- **tflite_flutter** — تشغيل نماذج TensorFlow Lite

### التخزين المحلي
- **hive** — قاعدة بيانات محلية سريعة
- **path_provider** — إدارة مسارات الملفات

### الصوت والوسائط
- **record** — تسجيل الصوت
- **video_player** — تشغيل الفيديو
- **flutter_svg** — عرض الرسومات المتجهة

### الأدوات المساعدة
- **get_it** — حقن التبعيات
- **connectivity_plus** — مراقبة الاتصال
- **crypto** — التحقق من سلامة الملفات
- **permission_handler** — إدارة الصلاحيات

## 🚀 البدء السريع

### المتطلبات

- Flutter SDK 3.24+
- Dart 3.4+
- Android Studio أو VS Code
- جهاز Android (API 21+) أو iOS (12+)

### التثبيت

```bash
# 1. استنساخ المشروع
git clone https://github.com/wael93/SingAIPureFlutter.git
cd SingAIPureFlutter

# 2. تثبيت الحزم
flutter pub get

# 3. توليد الكود التلقائي
flutter pub run build_runner build --delete-conflicting-outputs

# 4. تشغيل التطبيق
flutter run
```

### إعداد البيئة

```bash
# انسخ ملف الإعدادات
cp .env.example .env

# عدّل الإعدادات حسب احتياجك
nano .env
```
