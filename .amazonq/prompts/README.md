# 📝 SingAI Custom Prompts

هذا المجلد يحتوي على prompts مخصصة لتحسين كفاءة التطوير وتقليل استهلاك التوكنات.

## 📋 الملفات المتاحة

### 1. `singai-optimizer.md` (موصى به)
**الاستخدام**: للتطوير اليومي والمهام المتوسطة
**حجم التوكنات**: ~400 توكن
**متى تستخدمه**:
- بناء features جديدة
- تعديل كود موجود
- حل مشاكل تقنية
- طلب استشارات معمارية

**كيفية الاستخدام**:
```
@singai-optimizer اكتب service لتسجيل الصوت
```

### 2. `singai-quick.md` (للسرعة)
**الاستخدام**: للمهام السريعة والبسيطة
**حجم التوكنات**: ~200 توكن
**متى تستخدمه**:
- أسئلة سريعة
- كود بسيط
- مراجعة سريعة

**كيفية الاستخدام**:
```
@singai-quick كيف أضيف dependency جديدة؟
```

## 🎯 أفضل الممارسات

### ✅ افعل
- استخدم `@filename` لتحديد الملفات المطلوبة فقط
- اذكر الميزة المحددة التي تريد العمل عليها
- استخدم "كود فقط" إذا كنت لا تحتاج شرح
- راجع `@MEMORY_BANK.md` للأنماط الموجودة

### ❌ لا تفعل
- لا تطلب قراءة كل المشروع
- لا تطلب بناء تطبيق كامل في رد واحد
- لا تطلب tests (مرحلة MVP - توفير توكنات)
- لا تطلب شرح مطول للكود الواضح

## 🔧 أمثلة عملية

### مثال 1: إضافة feature جديدة
```
@singai-optimizer @MEMORY_BANK.md
اكتب service لتحويل النص إلى إشارات مع caching
```

### مثال 2: إصلاح bug
```
@singai-optimizer @lib/application/stt/stt_service.dart
الـ STT service يتوقف عند ملفات كبيرة، كيف أحل المشكلة؟
```

### مثال 3: سؤال سريع
```
@singai-quick
ما هو الـ pattern المستخدم للـ state management؟
```

### مثال 4: كود فقط
```
@singai-optimizer كود فقط
دالة لتنظيف النص العربي قبل المعالجة
```

## 🚀 Quick Commands المتاحة

عند استخدام `@singai-optimizer` أو `@singai-quick`:

- **"كود فقط"** → يعطيك الكود مباشرة بدون شرح
- **"خطة"** → يعطيك خطوات العمل بدون تنفيذ
- **"مراجعة"** → يحلل الكود الموجود
- **"تحسين"** → يقترح optimizations

## 📚 موارد إضافية

- `MEMORY_BANK.md` - معلومات شاملة عن بنية المشروع
- `README.md` - نظرة عامة على المشروع
- `.kiroignore` - ملفات مستبعدة من السياق

## 💡 نصائح لتوفير التوكنات

1. **حدد السياق**: استخدم `@filename` بدلاً من قراءة كل المشروع
2. **كن محدداً**: "اكتب دالة X" أفضل من "ساعدني في Y"
3. **استخدم Memory Bank**: معظم الأنماط موثقة فيه
4. **Quick Commands**: استخدم "كود فقط" عندما لا تحتاج شرح
5. **Prompt المناسب**: singai-quick للبسيط، singai-optimizer للمعقد
