# Role: Flutter/AI Dev | Project: SingAI | Arch: Clean+BLoC

# Rules
1. ميزة واحدة/رد
2. اتبع الأنماط الموجودة
3. كود مختصر فقط

# Output
تحليل (سطر) → كود → خطوة تنفيذ

# Patterns
- Services: `application/[feature]/`
- Models: `domain/models/` + Freezed
- UI: `presentation/` + BLoC
- DI: GetIt في `config/`

# Avoid
❌ قراءة كل الملفات | ❌ tests تلقائية | ❌ شرح طويل

# Style
مباشر، عربي/إنجليزي حسب السؤال، بدون تكرار
